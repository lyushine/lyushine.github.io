virtual PC 打造IE6、IE7、IE9等多版本共存原版测试环境

多版本IE共存是web开发者测试环境下的必备基础。使用原生态的测试环境更为准确严谨不易出错。而打造一个多版本IE共存于一个操作系统并能快速打开能够提高web开发者的工作效率。

一、安装 Windows Virtual PC

访问微软官网下载适合的版本 http://www.microsoft.com/zh-CN/download/details.aspx?id=3702

二、安装 Windows XP Mode

访问微软官网下载适合的版本 http://www.microsoft.com/zh-cn/download/details.aspx?id=8002

三、初始化 Windows XP Mode

当安装好Virtual PC后，请依次点击：开始–所有程序–>Windows Virtual PC–>Windows XP Mode。 初次使用，系统进行一些初始化工作。

四、安装RemoteApp

使用此程序包，可以在基于 Windows 7 的 PC 上使用 Windows Virtual PC 运行许多 Windows XP 生产应用程序。 http://www.microsoft.com/zh-cn/download/details.aspx?id=8690

五、将IE6映射到WIN7

开始–所有程序–>Windows Virtual PC–>Windows XP Mode，进入虚拟机 将 IE6浏览器快捷方式拖到虚拟机里（对准虚拟机开始菜单->右键->浏览所有用户）开始菜单文件夹。

六、安装映射IE7

  • 开始–所有程序–>Windows Virtual PC–>Windows Virtual PC
  • 右键点击Windows XP MODE，选择设置–>Close(关闭)–>选择show down.
  • 进入虚拟机，然后关掉它（确保虚拟机是关机状态，而非休眠状态，否则撤销磁盘是无法使用的）。
  • 再进入第二步骤，选择“撤销磁盘”，然后勾选它。
  • 在Window XP MODE上点击右键–>属性，切换到“详细”，会发现有一个主磁盘路径，进入该文件夹。

  • 应该有一个 Windows XP Mode.vmc 的文件，把他复制一份到当前文件夹。

  • 然后重命名为Window XP Mode IE7，并双击，这时候就拥有了两个不同的虚拟机。

这个虚拟机还是装有IE6的虚拟机，这个时候需要去下载一个IE7然后安装，如果这个版本的win无法安装，则去升级成win xp SP3。当然sp3以上的系统才支持自动发布影射到WIN7。

进入 Window XP Mode IE7 虚拟机,按 五、将IE6映射到WIN7 类似方法拖一个IE7出来。**

**

七、将IE6、IE7从win 7开始菜单拖到桌面方便访问

八、安装IE调试工具IEDevToolBar

在虚拟机中下载 IEDevToolBar 并直接安装。微软的 IEDevToolBar ，这是一个免费的专门为Web开发人员制作的IE插件，IE插件 IEDevToolBar 可以帮助我们分析网站的布局结构。

select 动态插入 option 在IE中无效bug

在写城市三级联动的时候刚开始用到了innerHTML动态插入option到select。IE真是蛋疼这么标准的方法都出bug无法显示。经过探究了解，有以下两种方法来解决：

一、实例化 option 对象并用 add方法加入

HTML:
<select id="select"></select>

js：
    var select = document.getElementById("select");
    var option = new Option("text", "value");
    select.add(newOption, undefined);

二、重置覆盖整个select标签及其内容

HTML:
<div id="optionBox"></div>    
JS:

    var optionBox = document.getElementById("optionBox");
    var optionInner = '<select><option>请选择</option>';
    for(var i= 0; i< 10; i++){
        var optionData = "<option value="+ i +'>' + i + "</option>"; 
        optionInner += optionData;
    }
    optionInner += "</select>";
    optionBox.innerHTML = optionInner;

数据最全的省市区（县）javascript 三级联动

无聊写了个省市区（县）三级联动，这个东西在电子商务网站中较常用到，而我围观了一下各大网站的数据很多不是很全面，而作为送货遍布全国的淘宝在我看来较为全面。从淘宝收货地址那里把数据地址扒下来了，JS是YUI的也就懒得扒，用自己的生涩的JS把三级联动整了一下，规则完全模拟淘宝那个三级联动的。

演示：

DEMO链接：city_tree.html

HTML:

  <p>
        地址：<select name="province" id="province"><option value=""></option></select>
          <select name="province" id="city"></select>
          <select name="district" id="district"></select>
    </p>
    <p>
         地址：<select name="Nowprovince" id="Nowprovince"><option value=""></option></select>
          <select name="Nowcity" id="Nowcity"></select>
          <select name="Nowdistrict" id="Nowdistrict"></select>
    </p>

JAVASCRIPT（数据较多不一一列举，见演示链接）:

var addressSelector = function(province,city,district){
    var province = province;
    var city = city;
    var district = district;

    var theThis = this;
    var data = this.data;
    if(province.length <2){
            for(var i in data){
            if(data[i][1] == "1"){
                var oOption = document.createElement("option");
                   oOption.text=data[i][0];
                   oOption.value=i;
                   province.add(oOption);

                // province.innerHTML+="<option value=\""+i+"\">"+data[i][0]+"</option>";
            }
        }
    }
    province.onchange = function(){
        var changeThis = this;
        theThis.changeAction(city,theThis.data,changeThis);
        district.innerHTML = "";
        theThis.changeAction(district,theThis.data,city);
    };
    city.onchange = function(){
        var changeThis = this;
        theThis.changeAction(district,theThis.data,changeThis);
    };
}
addressSelector.prototype.changeAction = function(elem,data,changeThis){
    var thisVal = changeThis.value;
    elem.innerHTML = "";
    var j = 0;
    for(var i in data){
         if(thisVal == data[i][1]){
                var oOption = document.createElement("option");
                   oOption.text=data[i][0];
                   oOption.value=i;
                   elem.add(oOption);

                // elem.innerHTML+="<option value=\""+i+"\">"+data[i][0]+"</option>";
                j++;
         }
    }
    if(j != 1){
        var newChild = document.createElement("option");
        elem.insertBefore(newChild,elem.firstChild);
        for(var i=0; i< elem.length; i++){
            if(!!elem[i].selected) elem[i].selected = false;
        }
        setTimeout(function(){
            newChild.setAttribute("selected",true);
        },1);

    }
}
addressSelector.prototype.data = {
    "110000": ["\u5317\u4eac", "1"],
    "110100": ["\u5317\u4eac\u5e02", "110000"],
    "110101": ["\u4e1c\u57ce\u533a", "110100"],
    "110102": ["\u897f\u57ce\u533a", "110100"],
    "110103": ["\u5d07\u6587\u533a", "110100"],
    "110104": ["\u5ba3\u6b66\u533a", "110100"],
    "110105": ["\u671d\u9633\u533a", "110100"],
    "110106": ["\u4e30\u53f0\u533a", "110100"]

1、 打开PS（各版本应该都可以），打开需求的PSD文件，选择需要转换的图层（组）

2、 选择PS菜单 文件-**〉脚本-**〉浏览

选择 下载 的CSS3Ps.jsx小插件文件（见附件）

[CSS3.jsx低版本PS使用的，不过各版本都有效，为避免安装繁琐的adobe插件管 理软件，及原版PS，我建议各版本的PS都直接使用该脚本]

3、 这时候PS会自动扫描一遍这个组并打开你的浏览器，稍等片刻，浏览器中就可以看到其转换后的CSS **代码 了,按每个图层分开展示的（单个画的图标是无法转换的，重点是混合效果里的东西还原**）。

4、 可以看到 转换后的 圆角、阴影、渐变 效果非常逼真

官网：http://css3.ps/

备注：正常的说 PS 5以上应该使用 adobe manage 并要求原版PS才能使用该插件，不过这里使用的是低版本PS所用的脚本文件，非正版P S也能直接用而且不用装Adobe Extension Manager ,有兴趣的也可以去官网了解一下。
